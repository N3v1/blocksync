### Getting Started:

1. **Clone the Repository:**
   ```
   git clone https://github.com/yourusername/blocksync.git
   ```

2. **Install Dependencies:**
   ```
   cd blocksync
   npm install
   ```

3. **Run BlockSync:**
   ```
   npm start
   ```
