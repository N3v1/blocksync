### Getting Started:

1. **Clone the Repository:**
   ```
   git clone https://github.com/BlockSyncHQ/blocksync.git
   ```

2. **Install Dependencies:**
   ```
   cd blocksync
   npm install
   ```

3. **Run BlockSync:**
   ```
   npm start
   ```
4. **Run the app with Android Studio or VS Code Or the command line:**
   ```
   flutter pub get
   flutter run
   ```
